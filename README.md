## 🚀 Getting Started  

1. **Clone the repo:**  
   ```bash
   git clone https://github.com/machinelearningprodigy/backend-test.git
   cd backend-test
   ```  

2. **Install dependencies:**  
   ```bash
   npm install
   ```  

3. **Run the server:**  
   ```bash
   npm start
   ```  
   The server will start on `http://localhost:5000` (default port).
